Curriculum of this diploma

CSS Basics

  • Inline, internal, and external CSS (style, link)
  • Selectors, properties, and values (e.g., color, background, font-size).
  • The box model (margin, padding, border).

CSS Layout

  • Positioning (relative, absolute, fixed, and sticky).
  • Display (block, inline-block, inline) and float properties.
  • Flexbox: creating flexible layouts with display: flex.
  • Grid: two-dimensional layouts with display: grid.

Responsive Design

  • Media queries for different screen sizes (mobile-first approach)
  • Units: pixels, percentages, em, rem, and viewport units (vh, vw).
  • Responsive images (max-width: 100%) and responsive typography.

CSS Frameworks

  • Introduction to CSS frameworks like Bootstrap or Tailwind CSS to speed up layout and design.
