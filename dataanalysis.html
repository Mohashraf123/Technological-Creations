Curriculum of this diploma

Mathematics & Statistics

  • Basics of Probability
  • Descriptive Statistics
  • Inferential Statistics

Data Types & Structures

  • Understanding different data types
  • Data structures: lists, dictionaries, data frames

Data Analysis Tools

  • Overview of tools like Excel, Google Sheets, and SQL basics

Python/R Basics

  • Control structures (loops, conditionals)

Data Manipulation

Data Visualization

  • Model Evaluation

Description:

Technoligical-Creations introduces the most powerful Data Analysis diploma that qualifies you to be a Data scientist - Data Engineer - Machine Learning Engineer.

We take you on a 6-month journey in the field of Data Analysis, during which you study and delve into everything related to this field, and study the following:
